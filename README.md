# poisonivy
Check the website on how to install and run the program

This is the vanilla version on GitHub. Derivatives must be clearly labeled as such.
In this game you find yourself in a lush or sometimes barren jungle. You need to craft and survive gathering your food by fishing, hunting, foraging or farming etc. There is no end but I add lore all the time. There is no permadeath like games have these days. You have 15 slots of savegames. Just take it easy. Sometimes I call it smelling the flowers.
