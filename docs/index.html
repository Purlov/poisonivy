User name Purlov and game name Jungle Hermit acquired on 2025-12-19.
In this game you find yourself in a lush or sometimes barren jungle. You need to craft and survive gathering your food by fishing, hunting, foraging or farming etc. There is no end but I add lore all the time. There is no permadeath like games have these days. You have 15 slots of savegames. Just take it easy. Sometimes I call it smelling the flowers.

Install - You only need the Windows command line for this, see Linux install further down below

winget install -e --id Python.Python.3.13 (by specifying the version "py -3.13", you can have multiple versions installed)

py -3.13 -m ensurepip --upgrade

py -3.13 -m pip install -U pygame

py -3.13 -m pip install -U numpy

py -3.13 -m pip install -U pillow

Run the game after agreeing to the blue license note below

Download the latest Source code (zip). It is located under Assets. New versions don't necessarily work with old save games. Save games are located in the folder "save". For backup purposes.

py -3.13 -m jungle (in the folder where you extracted the source code)
