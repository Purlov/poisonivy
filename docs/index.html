This is the vanilla version on github. Derivatives must be clearly labeled as such.
In this game you find yourself in a lush or sometimes barren jungle. You need to craft and survive gathering your food by fishing, hunting or foraging. There is no end but I add lore all the time. There is no permadeath like games have these days. You have 15 slots of savegames. Sometimes I call it smelling the flowers.

Install

winget install -e --id Python.Python.3.13 (by specifying the version "py -3.13", you can have multiple versions installed)

py -3.13 -m ensurepip --upgrade

py -3.13 -m pip install -U pygame

py -3.13 -m pip install -U numpy

py -3.13 -m pip install -U pillow

Run the game

Download the latest Source code (zip). It is located under Assets

py -3.13 -m poison (in the folder where you extracted the source code)
