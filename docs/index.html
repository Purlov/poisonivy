Basically a gladiator simulator. Here's some stream of consciousness crap that may not all exist:
In this game you are a gladiator in your team of computer gladiators. They fight automatically and you fight on their side. The team leadership issues you equipment and spells and also own money which can be used to buy more equipment or go to a bar with the computer gladiators from your team or others'. The more they are friends the less they attack you on the field or in the case that they are in your team they get a morale boost. You are a good fighter so sometimes leadership asks for your help. Or just be a laid-back dark elf frost mage behind the lines and enjoy the automatic fights. What if you can block somebody permanently into an ice cube? But then both of you are doing nothing. You can also use the extra money to do big special attacks.

Install

Download python 3.13 (version important) from python.org / choose to Add python.exe (py) to PATH

py -m pip install -U pygame --user

py -m pip install -U thorpy --user

py -m pip install -U numpy --user

py -m pip install -U pillow --user

The same should work in Linux and Mac as well (version might be less important)

Run the game

Download the latest Source code (zip) old save files do not usually work in new versions
